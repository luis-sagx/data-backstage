# Overview — Plantilla Data Backstage

Esta plantilla sirve como ejemplo sencillo para integrarse en Backstage.

- Genera un componente con metadatos básicos (nombre, descripción, owner).
- Incluye documentación preparada para TechDocs (la carpeta `docs/`).

Uso rápido:

1. Importa este repositorio en Backstage (añadiendo la plantilla al catálogo si es necesario).
2. Abre la sección "Create" / "Templates" en Backstage y busca "Plantilla Data Backstage".
3. Rellena el formulario con `name`, `description`, y `owner`.

Después de crear el componente, podrás ver su TechDocs apuntando a la carpeta `docs/`.
